## 🎨 界面功能

### 主要区域

#### 1. 标题栏
- 显示程序名称和版本
- 简短功能说明

#### 2. 状态指示器
- ● 未运行（灰色）
- ● 运行中（绿色）
- ● 已停止（灰色）

#### 3. 控制按钮
- **▶ 启动检测** - 开始运行检测系统
- **⏸ 停止检测** - 停止运行

#### 4. 快捷操作
- **🔍 测试OCR** - 运行OCR功能测试
- **⏱ 测试倒计时** - 测试倒计时识别
- **🔧 系统诊断** - 运行完整系统诊断
- **🗑 清空日志** - 清空日志显示区域

#### 5. 配置选项
- **使用OCR识别** - 开关OCR文字识别
- **调试模式** - 开关调试日志

#### 6. 运行日志
- 实时显示程序运行日志
- 自动滚动到最新日志
- 带时间戳的日志记录

---

## 📝 使用流程

### 基本使用

1. **启动GUI**
   ```
   双击 start_gui.bat
   ```

2. **配置选项**（可选）
   - 勾选"使用OCR识别"（推荐）
   - 需要调试时勾选"调试模式"

3. **启动检测**
   - 点击"▶ 启动检测"按钮
   - 观察日志确认启动成功

4. **开始游戏**
   - 程序在后台自动检测
   - 阵亡时自动切换到抖音
   - 复活时自动切回游戏

5. **停止程序**
   - 点击"⏸ 停止检测"按钮
   - 或直接关闭窗口

### 测试功能

在启动检测前，可以先测试各项功能：

```
🔍 测试OCR → 确认OCR是否正常工作
⏱ 测试倒计时 → 确认能否识别倒计时数字
🔧 系统诊断 → 检查系统环境是否正常
```

## 🐛 故障排查

### 问题1：GUI无法启动

**错误信息**：
```
ModuleNotFoundError: No module named 'customtkinter'
```

**解决方案**：
```bash
pip install customtkinter pillow
```

### 问题2：界面显示异常

**解决方案**：
- 确保Windows系统版本支持
- 更新显卡驱动
- 尝试设置环境变量：
  ```bash
  set CTK_THEME=blue
  ```

### 问题3：日志不显示

**解决方案**：
- 检查是否启用了调试模式
- 尝试点击"清空日志"后重启
