## 🎯 完整工作流程

```
1. 实时检测游戏画面（8 FPS）
   ↓
2. OCR识别"返回于"文字
   ↓
3. 提取复活倒计时（如53秒）
   ↓
4. 检查抖音页面是否存在
   ├─ 存在 → 切换到该页面
   └─ 不存在 → 打开新页面
   ↓
5. 发送播放键，开始播放视频
   ↓
6. 等待复活倒计时...
   ↓
7. 倒计时剩余2秒时
   ↓
8. 暂停视频
   ↓
9. 切换回LOL游戏
   ↓
10. 标记为复活状态
   ↓
11. 继续检测，等待下次阵亡
```

## 🚀 使用说明

### 快速开始

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **安装Tesseract OCR**（推荐）
   - 下载：https://github.com/UB-Mannheim/tesseract/wiki
   - 勾选"Chinese Simplified"

3. **运行程序**
   ```bash
   python main.py
   ```

### 配置建议

```python
# config.py
USE_OCR = True              # 使用OCR（推荐）
CAPTURE_FPS = 8             # 检测帧率
CONFIRM_FRAMES = 2          # 确认帧数
COOLDOWN_SECONDS = 15       # 冷却时间
DEBUG_MODE = False          # 调试模式
```
