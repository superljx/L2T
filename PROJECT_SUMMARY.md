## 🏗️ 技术架构

### 模块化设计

```
main.py (主控制器)
    ├─ detector.py (检测器)
    │   └─ 使用: mss, opencv, numpy
    ├─ window_manager.py (窗口管理)
    │   └─ 使用: win32gui, psutil
    ├─ browser_controller.py (浏览器控制)
    │   └─ 使用: subprocess
    ├─ input_controller.py (输入控制)
    │   └─ 使用: pyautogui
    └─ utils.py (工具函数)
        └─ 提供: 日志、计时器、性能监控等
```

### 数据流

```
屏幕捕捉 → 亮度计算 → 状态判断 → 帧缓冲确认 → 冷却检查 → 执行切换
   ↓          ↓          ↓           ↓           ↓          ↓
  mss      opencv    阈值比较    连续N帧      时间戳    窗口+按键
```

### 关键算法

#### 1. 亮度检测算法

```python
# HSV色彩空间的V通道（亮度）
hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)
brightness = np.mean(hsv[:, :, 2]) / 255.0 * 100.0

# 支持两种模式：
# - average: 简单平均
# - weighted: 中心加权（中心权重高）
```

#### 2. 自适应阈值

```python
# 动态学习基准亮度（中位数）
baseline = np.median(brightness_history[-100:])

# 判断阵亡
is_dead = (baseline - current_brightness) > threshold
```

#### 3. 连续帧确认

```python
# 帧缓冲器
buffer = [True, True, True]  # 示例：3帧

# 全True才触发
if all(buffer):
    trigger_action()
```

## 🔧 技术选型

### 依赖库

| 库 | 版本 | 用途 | 理由 |
|-------|--------|------|------|
| mss | 9.0.1+ | 屏幕捕捉 | 最快的跨平台截图库 |
| opencv-python | 4.8.0+ | 图像处理 | 强大的CV库，HSV转换 |
| numpy | 1.24.0+ | 数值计算 | 高效数组操作 |
| pywin32 | 306+ | 窗口管理 | Windows API访问 |
| psutil | 5.9.0+ | 进程管理 | 跨平台进程查询 |
| pyautogui | 0.9.54+ | 输入模拟 | 简单易用的键鼠控制 |
| pillow | 10.0.0+ | 图像支持 | pyautogui依赖 |

### 为什么选择这些库？

#### mss vs PIL.ImageGrab
- **mss**: 60+ FPS，跨平台
- **PIL**: 10-20 FPS，仅Windows
- **选择**: mss（性能优势明显）

#### win32gui vs pygetwindow
- **win32gui**: 原生API，功能完整
- **pygetwindow**: 封装层，功能有限
- **选择**: win32gui（可靠性更高）

#### pyautogui vs pynput
- **pyautogui**: 易用，功能全面
- **pynput**: 更底层，需要更多代码
- **选择**: pyautogui（开发效率）

## ⚠️ 已知限制

### 技术限制

1. **全屏独占模式**
   - 问题：某些全屏游戏无法截图
   - 解决：使用窗口化全屏

2. **游戏画面差异**
   - 问题：不同分辨率/设置下亮度不同
   - 解决：自适应阈值 + 手动调参

3. **特殊场景干扰**
   - 问题：技能特效、闪光可能误判
   - 解决：连续帧确认 + ROI检测

### 功能限制

1. **仅支持Edge浏览器**
   - 原因：Windows 11预装
   - 扩展：可修改支持Chrome/Firefox

2. **仅支持亮度检测**
   - 原因：简单高效
   - 升级：可引入深度学习模型

3. **单一游戏支持**
   - 原因：针对LOL优化
   - 扩展：可配置化支持其他游戏
