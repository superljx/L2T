# 🎉 v1.2.0 更新说明 - 复活倒计时功能

## 🆕 新增功能

### 自动切回游戏 + 暂停视频

现在程序不仅能检测阵亡并切换到抖音，还能：
- ✅ **识别复活倒计时**：OCR提取"返回于"下方的倒计时数字
- ✅ **自动切回游戏**：倒计时结束前2秒自动切回LOL
- ✅ **暂停视频播放**：切回游戏前暂停抖音视频
- ✅ **实时显示倒计时**：日志中显示剩余复活时间

---

## 🎯 完整流程

```
1. 检测到阵亡（"返回于"文字）
   ↓
2. 提取复活倒计时（如15秒）
   ↓
3. 切换到Edge并播放抖音
   ↓
4. 等待复活倒计时
   ↓
5. 倒计时剩余2秒时
   ↓
6. 暂停抖音视频
   ↓
7. 切换回LOL游戏窗口
   ↓
8. 继续游戏！
```

## 🔧 技术实现

### 1. OCR倒计时提取

程序使用多种模式识别倒计时：

**模式1：文本匹配**
```python
"返回于 15"     # 匹配数字
"返回于15秒"    # 匹配数字
"复活 10"       # 匹配数字
```

**模式2：区域识别**
- 在"返回于"下方区域单独识别数字
- 使用纯数字识别配置提高准确率

### 2. 倒计时管理

```python
respawn_timer_start   # 倒计时开始时间
respawn_duration      # 倒计时总时长
is_waiting_respawn    # 是否正在等待复活

# 实时计算剩余时间
remaining = respawn_duration - (time.time() - respawn_timer_start)

# 提前2秒切回
if remaining <= 2:
    switch_back_to_game()
```

### 3. 窗口查找逻辑

自动查找LOL游戏窗口（支持多种标题）：
- "League of Legends"
- "英雄联盟"
- "LOL"

## 🐛 故障排查

### 问题1：倒计时识别不到

**原因**：
- OCR未安装或配置错误
- 游戏分辨率过低
- 死亡界面UI被遮挡

**解决**：
1. 运行 `python test_ocr.py` 测试OCR
2. 启用调试模式查看OCR结果：
   ```python
   DEBUG_MODE = True
   ```
3. 查看日志中的OCR识别文本

### 问题2：没有自动切回游戏

**原因**：
- 未识别到倒计时
- LOL窗口标题不匹配

**解决**：
1. 查看日志是否有"已设置复活倒计时"
2. 检查游戏窗口标题是否包含关键词
3. 手动测试：运行中按Alt+Tab查看窗口列表

### 问题3：切回游戏太早或太晚

**解决**：
修改 `main.py` 中的切回时机：
```python
# 查找这行代码
if remaining <= 2 and remaining > 0:

# 修改数字2为其他值
if remaining <= 3 and remaining > 0:  # 提前3秒
if remaining <= 1 and remaining > 0:  # 提前1秒
```
