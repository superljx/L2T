# ✨ v1.3.0 更新说明 - 智能页面管理

## 🆕 新增功能

### 智能抖音页面管理

程序现在能智能判断是否需要打开新页面：

**之前 (v1.2.x)**：
- 每次阵亡都打开新的抖音页面
- 导致Edge中堆积大量抖音标签页
- 浪费资源

**现在 (v1.3.0)**：
- ✅ 检查Edge中是否已有抖音页面
- ✅ 如果存在 → 直接切换到该页面
- ✅ 如果不存在 → 打开新的抖音页面
- ✅ 避免重复打开，节省资源

## 🔍 技术实现

### 新增方法

**`find_douyin_tab()`**
- 遍历所有Edge窗口
- 检查窗口标题是否包含"抖音"或"douyin"
- 返回抖音页面的窗口句柄

**`switch_to_douyin_or_open(url)`**
- 首先尝试查找现有抖音页面
- 如果找到 → 直接激活该窗口
- 如果未找到 → 打开新页面并等待加载

### 识别逻辑

```python
# 通过窗口标题识别
title = win32gui.GetWindowText(hwnd)

if "抖音" in title or "douyin" in title.lower():
    # 找到抖音页面
    return hwnd
```

### 容错机制

```python
# 打开新页面后重试5次查找
for _ in range(5):
    time.sleep(0.5)
    douyin_hwnd = self.find_douyin_tab()
    if douyin_hwnd:
        return self.activate_window(douyin_hwnd)

# 如果仍未找到，切换到Edge主窗口
return self.switch_to_edge()
```

## 🐛 故障排查

### 如果无法找到抖音页面

**可能原因**：
- 抖音页面标题不包含"抖音"或"douyin"
- Edge窗口标题异常

**解决方案**：
1. 启用调试模式查看窗口标题
2. 程序会自动降级到打开新页面

### 如果打开了多个抖音页面

**可能原因**：
- 多次手动打开了抖音
- 程序未能正确识别现有页面

**解决方案**：
- 手动关闭多余的抖音标签页
- 程序下次会自动找到剩下的那个
